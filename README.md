# 🌿 Emily's Plant Collection

A personal plant care website tracking 14 houseplants and outdoor plants in Adelaide, Australia.

**Live site:** https://emilylu123.github.io/MyGreens/

---

## Features

- **Card view** — sorted by light, water, or humidity needs
- **Location view** — grouped by room
- **Bilingual** — switch between Chinese and English
- **Plant detail modal** — care guide, seasonal notes, September plan, and tips for each plant
- **Growth log** — take or upload photos per plant, stored locally on device (IndexedDB)
- **Status indicators** — colour-coded dots for stable, recovering, propagating, temporary indoors, and outdoors

## Tech

Pure HTML, CSS, and JavaScript — no frameworks or dependencies. Photos stored locally via IndexedDB and never uploaded to any server.

## Local use

Just open `index.html` in a browser. Place plant photos in the `images/` folder.
